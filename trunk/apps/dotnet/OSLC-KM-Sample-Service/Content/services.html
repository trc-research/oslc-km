OSLC Adapter-Services

As a compilation of all methods provided by this adapter the following table lists a description and a call example for each OSLC-based method.

 

URL Description Service
http://localhost:31885/oslc/concepts/info Description of the vocabulary service provider using OSLC. All
http://localhost:31885/oslc/concepts/resourceShape Description of the vocabulary resource shape. Vocabulary Management
http://localhost:31885/oslc/concepts/list Full description of all concepts in the vocabulary. Vocabulary Management
http://localhost:31885/oslc/concepts/list?max=10 Full description of all concepts in the vocabulary filtering max values. Vocabulary Management
http://localhost:31885/oslc/concepts/list?from=10&to=12 Full description of all concepts in the vocabulary filtering from a concept (from) to another concept (to). Vocabulary Management
http://localhost:31885/oslc/concepts/concept/14837 Description of a concept using the ID as input parameter. Vocabulary Management
http://localhost:31885/oslc/concepts/pref?q=store Description of a concept using the preferred label as input parameter. Vocabulary Management
http://localhost:31885/oslc/concepts/std?q=stores Get or suggest the standard form a label. Vocabulary Management
http://localhost:31885/oslc/concepts/syns?q=stores Get or suggest the new concepts for a term. Vocabulary Management
http://localhost:31885/oslc/concepts/asoc?q=stores Get or suggest associated concepts for a term. Vocabulary Management
http://localhost:31885/oslc/concepts/assist Get writing assistance for a concept. Vocabulary Management
http://localhost:31885/oslc/concepts/slot?q=stores Get concepts for the next slot (writing assistance. Vocabulary Management
http://localhost:31885/oslc/serviceProvider/info Description of the service provider using OSLC. All
http://localhost:31885/oslc/artifacts/resourceShape Meta-description of the OSLC resource for knowledge management Knowledge Management
http://localhost:31885/oslc/artifacts/list List all artifacts Knowledge Management
http://localhost:31885/oslc/artifacts/list?full=true List all artifacts (full description) Knowledge Management
http://localhost:31885/oslc/artifacts/artifact?name=DOMAIN&type=DOMAIN Description of a specific artifact Knowledge Management
http://localhost:31885/oslc/kpis/info Description of the service provider using OSLC. All
http://localhost:31885/oslc/kpis/resourceShape Meta-description of the OSLC resource for quality services KPI Management
http://localhost:31885/oslc/kpis/list Default list of KPIs KPI Management
http://localhost:31885/oslc/kpis/list?idBlock=12 List of KPIs for a block (id) KPI Management
http://localhost:31885/oslc/kpis/kpi/1 Description of a KPI KPI Management
http://localhost:31885/oslc/kpis/kpi/1?idBlock=12 Description of a KPI within a block (id) KPI Management
http://localhost:31885/oslc/kpis/observations?text=this is a bad requirement Default KPIs for a textual requirement... KPI Management
http://localhost:31885/oslc/kpis/observations?text=this is a bad requirement&idBlock=12 Default KPIs for a textual requirement within block (id)... KPI Management
http://localhost:31885/oslc/kpis/observations?uri=http://localhost:31885/oslc/requirements/requirement/1 Default KPIs for a OSLC-based requirement... KPI Management
http://localhost:31885/oslc/kpis/observations?uri=http://localhost:31885/oslc/requirements/requirement/1&idBlock=12 Default KPIs for a OSLC-based requirement within block (id)... KPI Management
http://localhost:31885/oslc/kpis/observations?uri=https://localhost:9443/rm/resources/_0b7af6d99f45484f9d20db5b263c5e34&project=Interoperability Default KPIs for a OSLC-based requirement within the Jazz Platform... KPI Management
http://localhost:31885/oslc/patterns/groups List of pattern groups Patterns Management
http://localhost:31885/oslc/patterns/group/12 Get a pattern group Patterns Management
http://localhost:31885/oslc/patterns/patternsgroup/7 Get the patterns of a pattern group Patterns Management
http://localhost:31885/oslc/patterns/pattern/537 Describe a pattern Patterns Management
http://localhost:31885/oslc/patterns/pattern/537?pgid=7 Describe a pattern (including pattern group) Patterns Management
http://localhost:31885/oslc/requirements/resourceShape Meta-description of the OSLC resource for requirements management Requirement Management
http://localhost:31885/oslc/requirements/list Get all requirements Requirement Management
http://localhost:31885/oslc/requirements/requirement/1 Get a requirement by id Requirement Management
http://localhost:31885/oslc/requirements/similar?text=an example of similar requirement Find a similar requirement by text Requirement Management
http://localhost:31885/oslc/requirements/similar?uri=http://localhost:31885/oslc/requirements/requirement/1 Find a similar requirement by URI Requirement Management
http://localhost:31885/oslc/requirements/similar?uri=https://localhost:9443/rm/resources/_0b7af6d99f45484f9d20db5b263c5e34&project=Interoperability Find a similar requirement by URI in a Jazz Project Requirement Management
